# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-01-31

### Added
- Integrated Whiskey dog outline logo into the brand section
- Added theme-aware manual logo assets (Burgundy for Dark Mode, Orange for Light Mode)
- Updated site title to "Whiskey Labs"
- Improved navigation bar alignment and spacing

## [0.1.0] - 2026-01-31

### Added

- Initial release of Whiskey Labs website
- Next.js 16 with App Router and TypeScript
- Tailwind CSS v4 design system
- Dark burgundy color palette with orange accents
- 3D orbital visualization using Three.js
  - Central planet with highlight
  - Translucent orbital ellipses
  - Animated orbiting spheres with glow effects
  - Background star particles
- Dashed grid layout system
- Home page with manifesto section
- Contact page with split-pane form
- Light/dark theme switching
- Responsive design

### Design

- Inspired by Stitch mockup
- "Whiskey" in bold white, "Labs" in muted grey
- Fuzzy aura effects on 3D elements
- Space-like aesthetic with geometric grid
